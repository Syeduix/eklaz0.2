What's Consistent

The shared design system holding it all together

Different layouts, identical foundations. Every page uses the same colors, fonts, header, footer, and button treatments from the approved home page.

🎨

Colors

Coral primary, cream backgrounds, navy footer, and pastel accents — exactly as the home page.

🔤

Typography

Plus Jakarta Sans for UI, Caveat script for eyebrows, Fraunces serif added for editorial display.

🧭

Header & Nav

Identical sticky header, top utility bar, primary nav, and call-to-action button on every page.

📐

Footer

Dark navy footer with newsletter form, 4-column links, and contact info — consistent across all 8 pages.

Notes for Implementation

How to read this delivery

All images use coral/pastel gradient placeholders for now — these are designed to be swapped with real photography when the site is integrated into WordPress. Every section is built with the same styles.css so editing is centralised.
